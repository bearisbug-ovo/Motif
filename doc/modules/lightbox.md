# 大图浏览 / LightBox

> 涉及代码：`frontend/src/components/lightbox/` | `frontend/src/components/video/` | `frontend/src/stores/lightbox.ts`

---

## 需求摘要

### 大图浏览模式 [P0]

**触发：** 点击任意图片卡片

**整体布局：**

```
┌─────────────────────────────────────────────────┐
│  [返回]  [评分★]  [AI功能按钮组]  [移动][封面][工作区][文件管理器][删除]  │ ← 顶部操作栏
├─────────────────────────────────────────────────┤
│                                                 │
│                  主图展示区                       │
│             （放大交互区域）                       │
│                                                 │
├─────────────────────────────────────────────────┤
│     生成链指示器（常驻，有生成图时显示）              │
├─────────────────────────────────────────────────┤
│     底部预览条（仅本地图，动态滚动高亮）              │
└─────────────────────────────────────────────────┘
```

**顶部操作栏：** 与图片右键菜单保持功能一致，布局更合理

**底部预览条：**
- 始终只显示本地图（这是布局设计，不受来源类型筛选控制），评分和媒体类型筛选生效
- 无图集/人物上下文时（如从任务队列、工作区打开），显示传入的全部 items
- 动态滚动，当前本地原图高亮居中（若当前图是生成图，高亮其所属的本地原图）
- 点击缩略图 = 水平轴导航，直接跳转到该本地图

### 双轴导航模型

大图模式提供两种独立的切换方式——**本地图导航（水平轴）** 和 **生成链导航（垂直轴）**，所有筛选中评分和媒体类型筛选生效，来源类型筛选不影响切换（以保证生成链遍历完整）。

**水平轴 — 本地图导航：** 在本地图序列中切换，跳过所有生成图。

| 优先级 | 当前位置 | 下一张 | 终止条件 |
|--------|---------|--------|----------|
| 1 | 任意图（含生成图） | 当前图所属本地原图的同图集内下一张本地图（按 sort_order） | 图集末尾时跳至优先级 2 |
| 2 | 图集末尾 | 同人物下一个图集的第一张本地图 | 人物下无更多图集时跳至优先级 3 |
| 3 | 人物末尾 | 下一个人物的第一张本地图 | 所有人物遍历完毕时提示"已到最后一张" |

> 若当前图是生成图，先沿 parent_media_id 回溯到本地原图，再从该本地图位置执行水平导航。

**无上下文降级（扁平模式）：** 从任务队列、工作区等无图集/人物上下文的入口打开大图时，水平轴退化为在传入的 items 扁平数组中顺序切换（所有 items 视为同一概念图集），不执行跨图集/跨人物跳转。垂直轴（生成链）仍正常工作。随机探索模式同样使用扁平模式（保持随机排列顺序）。

**垂直轴 — 生成链导航：** 在当前本地图的生成链中深度优先遍历。

| 优先级 | 当前位置 | 下一张（↓） | 终止条件 |
|--------|---------|-------------|----------|
| 1 | 任意图 | 当前图的第一张直接子生成图 | 无子生成图时跳至优先级 2 |
| 2 | 生成图 | 同父图的下一张兄弟生成图 | 无更多兄弟时跳至优先级 3 |
| 3 | 无更多兄弟 | 逻辑回溯到父图，继续找父图的下一个兄弟（递归向上） | 回到本地原图（根节点）且无更多子节点时跳至优先级 4 |
| 4 | 生成链遍历完毕 | 自动切换到下一张本地图（等同水平轴 →） | 与水平轴终止条件一致 |

> **回溯说明**：优先级 3 是逻辑回溯，不切换显示。例如生成链 `原图A → 放大1 → 换脸1`，在换脸1 按 ↓，无子代 → 无兄弟 → 回溯到放大1 → 找放大1 的下一个兄弟"放大2" → 切换到放大2。若放大1 也无更多兄弟 → 回溯到原图A → 找原图A 的下一个子代"放大3"（若有） → 否则生成链遍历完毕 → 切换下一张本地图。

上一张（↑）反向遍历，规则对称。在本地原图（根节点）按 ↑ 切换到上一张本地图（等同水平轴 ←）。无生成图的本地图按 ↓ 直接切换下一张本地图。

> 生成链递归最大深度 10 层，超过时跳过深层子节点。

### 生成链指示器

常驻位于主图区与底部预览条之间，显示当前图在生成链中的位置。

*默认态（收起）：*
- 紧凑单行横向排列：`[本地原图] → [子1] [子2] ... → [孙1] ...`，缩略图小圆角（24×24），用连线和缩进表达层级
- 当前图高亮边框（primary 色），其余半透明
- 不同工作流类型用不同边框颜色区分（upscale=蓝, face_swap=紫, inpaint=琥珀, screenshot=绿）
- 无生成图时整行不显示（高度收为 0）
- 点击任意节点 = 跳转到该图（垂直轴导航）
- 滚轮在该区域上滚动 = 垂直轴生成链导航

*展开态（点击展开按钮）：*
- 展开为浮层水平树布局（floating overlay），横向 = 生成深度（列），纵向 = 同一父节点的子节点按创建时间从上到下排列
- 节点以缩略图（40×40）展示，底部叠加工作流类型标签，SVG 贝塞尔曲线连接父子节点
- 点击外部区域收起（click-outside-to-close）
- 支持在节点上右键"脱离生成链"
- 生成链递归最大深度 10 层，超过时折叠显示

### 沉浸模式（全屏浏览）

**触发：** 顶部操作栏沉浸模式按钮，普通大图模式和随机探索均可开启

**开启后：** 隐藏所有 UI（顶部栏、底部预览条、生成链指示器、左侧导航），仅保留图片全屏展示

**保留操作：**
- 本地图导航：键盘 `←` / `→`；手机左右滑动
- 生成链导航：键盘 `↑` / `↓`；手机上下滑动
- 评分：键盘 `1`-`5`；手机**双击**图片弹出评分浮层（点击星级后浮层消失）

**退出沉浸模式：**
- PC：鼠标右键弹出菜单 → 选择"退出沉浸模式"；或按 `Esc`
- 手机：两指同时触碰弹出菜单 → 选择"退出沉浸模式"（大图模式统一使用两指触碰替代长按弹菜单，避免与视频倍速手势冲突）

### 放大模式

**PC 端（鼠标）：**
- **触发：** 左键单击主图区 / 滚轮向上 / 双击
- **行为：** 不再使用覆盖层，直接对主图 CSS transform 缩放，以触发点为中心放大
- **缩放：** 滚轮在图片上以鼠标位置为中心缩放（1x ~ 8x），zoom factor = 1.15/tick
- **平移：** 已放大状态下鼠标移动即平移图片，鼠标在原图上的位置点与放大图对应位置精确重合；鼠标指针替换为 SVG 放大镜图标（CSS `cursor: url(...)` 原生渲染）；鼠标移出原图范围时隐藏放大图并恢复默认指针；无边界约束
- **双击：** 切换 fit ↔ 2x（带 0.3s 缓动动画）
- **退出：** 单击 / 右键 / Esc 退出放大模式（动画回到 fit-to-screen）
- **方向键（放大中）：** ← → ↑ ↓ 在放大模式下直接执行导航（水平/垂直轴），保持当前缩放比例不变
- **滚轮区域分离：** 黑边和缩略条区域滚轮切换图片，图片区域滚轮缩放
- **性能：** CSS transform + will-change + rAF 直接 DOM 操作，60fps

**移动端（触摸）：**
- **触发：** 双指捏合（pinch-to-zoom）放大图片
- **行为：** 图片以捏合中心为原点缩放（1x ~ 8x），支持双指平移
- **单指拖拽：** 放大状态下单指可拖拽平移浏览
- **退出：** 单击图片恢复原始大小；缩放接近 1x 时自动 snap 回原始尺寸
- **仅图片：** 视频不支持放大模式

**禁用时机：** 蒙版编辑器中禁用放大模式，改为画布整体缩放

### 视频大图模式 [P0]

**布局：** 同图片大图模式，主区域替换为独立 `<VideoPlayer>` 组件（双 `<video>` 元素 A/B 交替，消除切换黑闪）

**关闭行为：** 与图片大图一致 —— 点击视频画面内的 letterbox 黑边区域可关闭大图模式，点击视频内容区域不关闭

**播放器功能：**
- 播放/暂停、可点击/拖拽进度条（点击即跳转 + 拖拽 seek，含缓冲进度）、音量滑块、全屏、逐帧步进
- 倍速选择（0.5x--3x，PC 下拉菜单选择）
- 自动播放 + 静音策略：首次进入自动静音播放，用户手动取消静音后同会话内后续视频不再静音
- PC 键盘快捷键：空格（播放/暂停）、←/→（快退/快进 5s）、F（全屏）
- **视频模式键盘轴分配：** ← → 归 VideoPlayer（快退/快进），不触发水平轴导航（水平轴通过底部预览条/导航条/移动端滑动操作）；↑ ↓ 竖屏时用于生成链导航（如视频截图属于视频的生成子代），横屏时归 VideoPlayer（音量 ±10%）
- **移动端控件栏精简**：采用溢出菜单布局，控件栏仅显示「播放/暂停 | 音量 | 时间 | ··· | 横屏」，次要功能（倍速、逐帧、截图）收入 `···` 弹出菜单。移动端横屏按钮等同全屏（不再单独显示全屏按钮）
- **右键菜单静音切换**：视频大图模式的右键菜单（移动端双指触碰）包含「取消静音/静音」选项，方便移动端切换

**播放进度记忆：**
- `initialTime` 在 `loadedmetadata` 后 seek 到保存位置（距结尾 3s 内不恢复），显示"从 X:XX 继续播放"提示 2.5s
- `onProgressSave` 在暂停/定时 15s/离开视频时调用，位置变化 > 2s 才保存
- 播完自动清除进度（position=0）
- 进度存储在 DB，多设备共享

**移动端手势（Touch Arbiter 统一仲裁）：**

多种手势通过 `useTouchArbiter` 状态机统一仲裁，同一时刻仅允许一种手势生效：

```
idle → (touchstart) → pending
pending → (进度条 pointerdown) → seeking
pending → (横屏 + |dx|>5 && |dx|>|dy|) → seeking（拖拽 seek）
pending → (200ms 长按定时器，期间无水平移动) → speed_control
pending → (竖屏 + |dx|>5 && |dx|>|dy|) → swiping
pending → (touchend <300ms, 无移动) → tap → idle
seeking/swiping/speed_control → (touchend/pointerup) → idle
```

- **单击**（<300ms 无移动）：切换控件显示/隐藏
- **进度条拖拽**：pointerDown 立即 claim `seeking`，阻止其他手势
- **横屏水平拖拽视频区域**：claim `seeking`，拖拽距离占屏幕宽度比例映射到 min(duration, 600s) 进行 seek；拖拽开始时暂停视频避免反复缓冲，松开后恢复播放；拖拽期间强制显示进度条，松开后立即收起
- **手势优先级**：pending 状态下任何水平移动（>1px）立即取消长按定时器，确保拖拽 seek 优先于倍速触发
- **长按**（200ms，期间无移动）：进入 2x 倍速播放（claim `speed_control`）
- **长按中左右滑动**：调节倍速 0.5x--3x，步进 0.25x
- **长按后松开**：恢复原速 1x
- **长按后上滑再松开**：锁定当前倍速（顶部显示倍速 + 锁图标）
- **两指同时触碰**：弹出右键菜单（替代长按弹菜单，避免与倍速长按冲突；仅大图模式，其他页面仍用长按）
- **竖屏时左右滑动**（claim `swiping`）：由 LightBox 处理，本地图导航（水平轴）
  - **速度阈值**：velocity > 0.3 px/ms 且 |dx| > 20px（快速轻扫）
  - **距离阈值**：|dx| > 60px（慢拖兜底）
  - 满足任一即触发切换

## 前端架构

### LightBox 组件树

```
LightBox.tsx                    主容器（createPortal → body，全屏覆盖 bg-black）
├── LightBoxTopBar.tsx          顶部操作栏（评分、AI功能、封面、工作区、删除等）
├── LightBoxMedia.tsx           主图/视频展示区
│   ├── <img>                   图片模式（useImageZoom hook 管理放大交互）
│   └── <VideoPlayer>           视频模式
├── LightBoxContextMenu.tsx     右键菜单（封面、AI子菜单、静音、工作区、移动、生成链、评分、删除）
├── ChainIndicator.tsx          生成链指示器（收起态单行 / 展开态水平树浮层）
└── ThumbnailStrip.tsx          底部缩略图条（仅本地图，滚动高亮当前图）
```

**统一输入管理：** `hooks/useLightboxInput.ts` 统一处理键盘和鼠标/触摸事件的映射逻辑。

### VideoPlayer 架构

**双 video 元素 A/B 交替消除切换黑闪：**
- 使用两个 `<video>` 元素交替显示
- `requestVideoFrameCallback` 等新视频第一帧渲染后瞬间 swap z-index
- 非活跃 slot 加 `invisible` 防止残影（不同宽高比视频切换时旧帧从边缘露出）
- 视频区域用 `flex-1` 自适应 LightBox 剩余空间，不用固定 calc 高度

**Props：**
- `src`、`poster`、`autoPlay`、`initialMuted`、`initialTime?`（播放恢复位置）
- `onMutedChange`、`onProgressSave?`（播放进度保存回调）、`onScreenshot`
- `isLandscape`、`onLandscapeChange`、`touchArbiter?`、`onContextMenu?`
- 通过 `forwardRef` 暴露 `VideoPlayerHandle`（`toggleMute()`），供 LightBox 右键菜单调用

**子组件/Hooks：**

| 模块 | 职责 |
|------|------|
| VideoControls | 控件栏：进度条、音量、倍速、逐帧、截图、全屏/横屏 |
| VideoGestureLayer | 手势层：接收 touchArbiter，处理长按倍速/横屏拖拽 seek |
| useControlsAutoHide | PC 鼠标 3s 无移动隐藏控件；移动端单击切换 |
| useTouchArbiter | 统一手势仲裁状态机（idle → pending → seeking/swiping/speed_control → idle） |
| useSpeedControl | 长按倍速控制（200ms → 2x，左右滑调速，上滑锁定） |
| useOrientationMode | 横屏模式（requestFullscreen → orientation.lock('landscape')） |
| SpeedIndicator | 顶部居中倍速提示条（显示当前倍速 + 锁定图标） |

### Store 职责（stores/lightbox.ts）

管理大图浏览的全部状态：
- 双轴导航状态（当前项、本地图序列、生成链树）
- navigateH(+1/-1) 水平轴导航、navigateV(+1/-1) 垂直轴导航
- flatMode 标记（无上下文时降级为扁平数组顺序切换）
- sessionUnmuted 静音策略状态

## 交互规则

### 键盘快捷键

| 按键 | 图片模式 | 视频模式 |
|------|---------|---------|
| `←` / `→` | 本地图导航（水平轴） | 快退/快进 5s（不触发水平轴导航） |
| `↑` / `↓` | 生成链导航（垂直轴） | 竖屏=生成链导航；横屏=音量 ±10% |
| `Esc` | 退出大图/退出沉浸/退出放大 | 退出大图/退出沉浸 |
| `1`-`5` | 快速评分 | 快速评分 |
| `0` | 清除评分 | 清除评分 |
| `空格` | -- | 播放/暂停 |
| `F` | -- | 全屏切换 |

> 焦点在 input/textarea 内时方向键不拦截。放大模式下方向键导航但保持缩放状态。

### 鼠标操作（PC）

| 操作 | 行为 |
|------|------|
| 单击主图 | 进入放大模式（以点击位置为中心放大到 2x，带动画） |
| 单击黑边区域 | 退出大图模式（不触发放大） |
| 单击（已放大时） | 退出放大模式（动画回到 fit） |
| 双击主图 | 切换 fit ↔ 2x（带动画） |
| 滚轮（图片上） | 生成链导航（垂直轴）；若无生成链则以鼠标为中心缩放（1x ~ 8x） |
| 滚轮（黑边区域） | 生成链导航（垂直轴） |
| 滚轮（底部预览条） | 横向滚动预览条本身（不触发导航） |
| 滚轮（生成链指示器） | 生成链导航（垂直轴） |
| 滚轮（弹窗覆盖层） | 不触发导航（`[role="dialog"]` 内的滚轮事件被屏蔽） |
| 鼠标移动（已放大时） | 图片跟随鼠标平移，鼠标指针显示为放大镜图标 |
| 右键（已放大时） | 退出放大模式 |

### 左右切换导航条（PC）

- 左右边缘各显示一个全高导航长条（`w-28`），hover 时显示半透明背景（`bg-white/20`）和方向箭头图标
- 鼠标指针替换为 SVG 方向箭头（CSS `cursor: url(data-uri)` 原生渲染）
- 到达首张/末张时对应方向导航条不显示
- 仅 PC 端显示，移动端使用滑动手势切换
- 导航条点击 = 本地图导航（水平轴）

### 轴切换视觉反馈

- 水平轴切换：底部预览条高亮闪烁一次（`opacity 0.6 → 1`，150ms）
- 垂直轴切换：生成链指示器高亮闪烁一次，同时当前节点边框 pulse 动画（150ms）
- 图片本身无方向性过渡动画，直接替换（crossfade 也不加，避免累积延迟）

### 预加载策略

`LightBox.tsx` 内 useEffect 监听 `item?.id` 变化，按优先级预加载前后共 8 张图片（偏移量 +1, -1, +2, -2, +3, -3, +4, +5，近处优先、前进方向偏重）。图片通过 `new Image()` 触发浏览器缓存，视频通过 `<link rel="preload">` 预获取。

### 视频控件布局

| 布局模式 | 场景 | 样式 |
|---------|------|------|
| `bottom-bar` | PC 非全屏 | 视频下方固定 |
| `overlay` | 横屏/全屏 | 底部悬浮渐变背景 |

**自动隐藏（useControlsAutoHide）：** PC 鼠标 3s 无移动隐藏控件；移动端单击切换。
