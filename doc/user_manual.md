## 1. 安装与启动

### 1.1 环境要求

| 项目 | 要求 |
|---|---|
| 操作系统 | Windows 10/11 |
| Python | 3.11（后端虚拟环境） |
| Node.js | 18+（前端构建） |
| ComfyUI | 本地安装（AI 功能需要，纯浏览管理可不装） |
| 显卡 | 8GB+ VRAM（AI 功能推荐） |

### 1.2 首次安装

```bash
# 1. 克隆或下载项目
git clone https://github.com/your-username/Motif.git
cd Motif

# 2. 安装后端依赖
cd backend
python -m venv venv
venv\Scripts\pip.exe install -r requirements.txt
venv\Scripts\alembic.exe upgrade head
cd ..

# 3. 安装前端依赖
cd frontend
npm install
cd ..
```

### 1.3 启动方式

**方式一：一键启动（推荐）**

双击项目根目录的 `start.bat`。脚本会：

1. 检测是否已有 Motif 实例在运行（防止重复启动）
2. 构建前端（`npm run build`）
3. 启动后端（端口 8000，同时 serve 前端页面）
4. 启动 ComfyUI（如已配置，端口 8188）
5. 自动打开浏览器（`http://localhost:8000`）

启动后进入交互模式：
- 按 `R` — 重新构建前端 + 重启后端
- 按 `Enter` — 停止所有服务并退出

**方式二：手动启动**

打开两个终端窗口分别启动：

```bash
# 终端 1 — 构建前端 + 启动后端
cd frontend && npm run build && cd ../backend
venv\Scripts\uvicorn.exe main:app --host 0.0.0.0 --port 8000

# 终端 2 — ComfyUI（可选）
"D:\ai\ComfyUI-aki-v1.6\python\python.exe" "D:\ai\ComfyUI-aki-v1.6\ComfyUI\main.py" --port 8188 --lowvram
```

> 后端自动 serve `frontend/dist/` 目录，通过 `http://localhost:8000` 统一访问。

### 1.4 局域网访问

Motif 绑定 `0.0.0.0`，同一局域网下的其他设备可通过 `http://<电脑IP>:8000` 访问。在手机浏览器中可添加到主屏幕作为 PWA 应用使用（standalone 模式需在 Chrome flags 中将该地址标记为安全来源）。

## 2. 界面总览

### 2.1 页面结构

| 页面 | 说明 | 导航方式 |
|---|---|---|
| 人物库（首页） | 所有人物的卡片网格 | 侧边栏点击"人物库" |
| 人物主页 | 某人物的图集区 + 散图区 | 点击人物卡片进入 |
| 图集详情 | 某图集内的媒体列表 | 点击图集卡片进入 |
| 大图浏览 | 全屏查看图片/视频 | 点击任意媒体缩略图 |
| 任务队列 | AI 任务管理 | 侧边栏点击"任务" |
| 工作区 | 跨页面媒体收藏书架 | 侧边栏点击"工作区" |
| 回收站 | 已删除的媒体 | 侧边栏点击"回收站" |
| 设置 | 系统配置 | 侧边栏点击"设置" |

### 2.2 侧边栏

左侧侧边栏提供全局导航：
- 固定宽度展开（176px），始终显示图标 + 文字
- 显示 ComfyUI 连接状态指示灯（绿色=已连接，红色=未连接）
- 移动端显示为底部导航栏

### 2.3 筛选栏

每个列表页顶部都有筛选栏（FilterBar），提供：
- **排序** — 导入时间 / 评分 / 名称
- **评分筛选** — 选择 ≥ / = / ≤ 某个星级
- **媒体类型** — 全部 / 图片 / 视频
- 各页面有独立的默认筛选配置，可在设置页修改

## 3. 人物管理

### 3.1 创建人物

1. 在人物库页面，点击右上角 **"+ 新建人物"** 按钮
2. 输入人物名称
3. 确认创建

### 3.2 人物卡片

人物卡片显示：
- 封面图（可自定义）
- 人物名称
- 平均评分（有评分媒体的均值）
- 媒体数量

**卡片网格缩放：** 按住 `Ctrl` + 滚轮可调整卡片列数（3–12+ 列）。

### 3.3 编辑人物

- 右键人物卡片 → 弹出菜单
- 可执行：重命名、设置封面、删除

### 3.4 人物主页

点击人物卡片进入人物主页，分为两个区域：

- **图集区**（上方）— 该人物的所有图集，以卡片网格展示
- **散图区**（下方）— 未归入任何图集的媒体

## 4. 图集管理

### 4.1 创建图集

1. 在人物主页，点击 **"+ 新建图集"**
2. 输入图集名称
3. 图集自动归属当前人物

### 4.2 图集操作

- **重命名** — 右键图集卡片 → 重命名
- **设置封面** — 在大图浏览中右键图片 → 设为图集封面
- **删除图集** — 右键 → 删除（媒体回到散图区，不会被删除）

### 4.3 图集详情

点击图集卡片进入详情页，支持两种布局：
- **网格模式** — 等高行或等宽列排列
- **行模式** — 单行横向排列，可调节行高

### 4.4 移动媒体到图集

在大图浏览中右键图片 → "移动到图集" → 选择目标图集。

## 5. 媒体导入

### 5.1 文件夹导入

1. 点击页面中的 **"导入"** 按钮
2. 选择本地文件夹
3. Motif 扫描文件夹（含子文件夹），显示发现的图片和视频
4. 为每个子文件夹指定归属的人物和图集
5. 确认导入

**重要：** Motif 不复制文件，只记录文件路径。请勿在导入后移动或删除源文件。

### 5.2 剪贴板导入

复制图片后，在 Motif 中按 `Ctrl+V`，图片会保存到 AppData 的 clipboard 目录并导入。

### 5.3 支持的格式

- **图片** — JPG、PNG、WebP、GIF、BMP
- **视频** — MP4、MOV、AVI、MKV、WebM

## 6. 大图浏览（LightBox）

点击任意媒体缩略图即可进入大图浏览模式。

### 6.1 基本操作

| 操作 | PC | 移动端 |
|---|---|---|
| 打开 | 点击缩略图 | 点击缩略图 |
| 关闭 | `Esc` / 点击黑边区域 | 点击关闭按钮 |
| 上一张 | `←` 键 / 滚轮（黑边区域）/ 点击左侧导航条 | 左滑 |
| 下一张 | `→` 键 / 滚轮（黑边区域）/ 点击右侧导航条 | 右滑 |
| 评分 | `1`–`5` 键 | 双击弹出评分浮层 |

**左右导航条（PC）：** 左右边缘各有一个全高导航区域，鼠标悬停时显示半透明背景和方向箭头，光标变为对应方向箭头。点击即可切换图片。到达首张/末张时对应方向的导航条自动隐藏。

### 6.2 放大模式（仅图片）

放大模式提供无损细节查看：

**进入放大：**
- 单击图片 → 以点击位置为中心放大到 2x
- 滚轮向上（图片上）→ 逐级放大
- 双击 → 切换 fit ↔ 2x

**放大中操作：**
- **鼠标移动** — 图片跟随鼠标平移，鼠标位置与放大图对应位置精确重合
- **滚轮** — 以鼠标为中心继续缩放（1x–8x）
- 鼠标指针自动替换为放大镜图标

**退出放大：**
- 单击图片
- 右键
- `Esc`
- 切换到其他图片

**超出隐藏：** 鼠标移出原图范围（移到黑边区域）时，放大图自动隐藏，鼠标恢复默认箭头；移回图片区域时放大图重新出现。

**移动端放大：**
- 双指捏合放大（1x–8x）
- 单指拖拽平移
- 单击恢复原始大小

> 视频不支持放大模式。

### 6.3 沉浸模式

点击顶部操作栏的沉浸模式按钮，隐藏所有 UI 元素，仅保留图片全屏展示。

沉浸模式中保留操作：
- `←` / `→` 切换图片
- `1`–`5` 评分
- `Esc` 或右键菜单退出沉浸模式

### 6.4 视频模式

视频媒体在大图浏览中使用专用播放器：

- **播放/暂停** — 空格键或点击
- **调速** — 0.5x / 1x / 1.5x / 2x / 3x
- **逐帧** — `,` 上一帧 / `.` 下一帧（暂停状态）
- **截图** — 捕获当前帧保存为图片
- **设为封面** — 将当前帧设为所属图集或人物的封面
- **静音** — 右键菜单可切换

### 6.5 右键菜单

在大图浏览中右键图片/视频弹出操作菜单：

- 设为封面（人物/图集）
- AI 功能（放大 / 换脸 / 局部修复）— 仅图片
- 静音 — 仅视频
- 添加到工作区
- 移动到图集
- 在资源管理器中打开
- 查看生成链 / 脱离生成链
- 评分
- 删除

### 6.6 底部缩略图条

大图底部显示当前上下文的缩略图条，当前图片高亮，点击可跳转。

## 7. 评分与筛选

### 7.1 评分

- **大图浏览中** — 键盘 `1`–`5` 快速评分；或通过右键菜单
- **缩略图上** — 悬停时显示星级，点击评分
- 再次点击相同星级可取消评分

### 7.2 筛选

每个列表页的筛选栏支持：

| 筛选项 | 选项 |
|---|---|
| 评分 | ≥1★ / ≥2★ / ≥3★ / ≥4★ / =5★ / ≤N★ / 无评分 |
| 媒体类型 | 全部 / 图片 / 视频 |
| 排序 | 导入时间 / 评分 / 文件名 |
| 排序方向 | 升序 / 降序 |

### 7.3 筛选默认值

在设置页可为每个页面配置默认筛选条件，每次打开页面自动应用。

## 8. AI 功能

> AI 功能需要本地运行 ComfyUI 并正确配置模型。侧边栏会显示 ComfyUI 连接状态。

### 8.1 高清放大

将图片放大 2x 或 4x，提升分辨率和细节。

1. 在大图浏览中右键图片 → AI 功能 → 高清放大
2. 选择放大倍率
3. 提交任务

结果图片自动关联为原图的子图（生成链）。

### 8.2 换脸

将图片中人物的面部替换为参考人脸，保持姿势、服装和背景不变。

**单张换脸：**
1. 右键目标图片 → AI 功能 → 换脸
2. 选择人脸参考图
3. 提交任务

**批量 AI（换脸/放大/图生图/预处理）：**
1. 通过图集右键菜单 → AI 批量 → 选择类别，或多选图片后点击工具栏「AI 批量」按钮
2. 选择具体工作流，配置共享参数（如人脸参考图）
3. 提交批量任务（仅处理本地图和截图，跳过 AI 生成图）
4. 结果逐张加入原图的生成链，可在大图模式查看

### 8.3 局部修复（Inpainting）

对图片的指定区域进行 AI 重绘。

1. 右键图片 → AI 功能 → 局部修复
2. 打开蒙版编辑器
3. 用画笔涂抹需要修复的区域（绿色遮罩）
4. 输入提示词描述期望的效果
5. 提交任务

**蒙版编辑器操作：**
- 画笔大小：`[` / `]` 调整
- 撤销：`Ctrl+Z`
- 重做：`Ctrl+Shift+Z`
- 橡皮擦：切换擦除模式

### 8.4 生成链

每次 AI 生成的图片会记录来源关系：

- 在大图浏览中，右侧面板可展开"生成链"
- 树形结构展示原图 → 放大 → 换脸 → 修复的完整衍生路径
- 点击节点可跳转到对应图片
- "脱离生成链"可断开与父图的关联

## 9. 任务队列

### 9.1 概述

所有 AI 任务通过任务队列严格串行执行（一次只向 ComfyUI 提交一个任务），确保显存不溢出。

### 9.2 任务状态

| 状态 | 含义 |
|---|---|
| 等待中 (pending) | 排队等待执行 |
| 执行中 (running) | 正在 ComfyUI 中处理 |
| 已完成 (completed) | 执行成功，可查看结果 |
| 失败 (failed) | 执行出错，可查看错误信息并重试 |

### 9.3 任务操作

- **查看** — 在任务队列页面查看所有任务
- **排序** — 拖拽调整等待中任务的执行顺序
- **取消** — 取消等待中的任务
- **重试** — 对失败任务重新提交
- **查看结果** — 已完成任务可直接跳转到生成的图片

## 10. 工作区

工作区是一个持久化的媒体引用书架，用于跨人物/图集收集素材。

- **添加** — 在大图浏览中右键 → "添加到工作区"
- **移除** — 在工作区中右键 → "从工作区移除"
- **排序** — 拖拽调整顺序
- **上限** — 最多 100 条引用
- 工作区数据持久化存储在数据库中，重启不丢失

## 11. 回收站

### 11.1 删除与恢复

- 删除媒体时执行"软删除"，媒体移入回收站
- 在回收站页面可以恢复或永久删除
- 本地图片的永久删除只删数据库记录，不删源文件
- 生成图和截图的永久删除会删除物理文件

### 11.2 自动清理

回收站默认 30 天后自动永久删除（可在设置中修改，设为 0 禁用自动清理）。

## 12. 设置

在设置页可配置：

| 设置项 | 说明 | 默认值 |
|---|---|---|
| ComfyUI 地址 | ComfyUI API 端点 | `http://127.0.0.1:8188` |
| AppData 路径 | 数据存储目录（缩略图、生成图等） | `backend/appdata/` |
| 缩略图大小 | 缓存缩略图的边长 | 400px |
| 回收站保留天数 | 自动清理天数（0=永不自动清理） | 30 天 |
| 任务超时 | ComfyUI 单任务执行超时 | 10 分钟 |
| 各页面默认筛选 | 为每个页面配置默认排序和筛选条件 | — |

**AppData 路径迁移：** 修改 AppData 路径时，Motif 会将现有数据迁移到新位置。

## 15. 常见问题

### ComfyUI 连接失败

1. 确认 ComfyUI 已启动并运行在配置的端口（默认 8188）
2. 在设置页检查 ComfyUI 地址是否正确
3. 确认防火墙未阻止连接

### 图片显示"文件丢失"

Motif 不复制文件，只存储路径引用。如果源文件被移动、重命名或删除，Motif 会显示丢失标记。请将文件恢复到原始位置。

### AI 任务一直排队/执行失败

1. 检查 ComfyUI 连接状态（侧边栏指示灯）
2. 确认所需模型已正确放置（见 [CLAUDE.md](../CLAUDE.md#comfyui-环境) 的模型路径列表）
3. 检查显存是否充足（推荐 8GB+）
4. 在任务队列页面查看失败任务的错误信息

### 缩略图不显示

缩略图缓存在 AppData 目录下。如果首次导入大量图片，缩略图会在后台逐步生成。刷新页面即可看到新生成的缩略图。

### 局域网其他设备无法访问

1. 确认 Motif 运行的电脑和访问设备在同一局域网
2. 检查 Windows 防火墙是否放行了端口 8000
3. 使用电脑的局域网 IP（非 `localhost`），如 `http://192.168.1.100:8000`

### 如何备份数据

Motif 的所有数据存储在 AppData 目录中（默认 `backend/appdata/`），备份该目录即可。数据库文件为 `db/main.sqlite`。

> **注意：** 备份不包含源媒体文件本身（Motif 只存路径引用），请单独备份您的媒体文件。
